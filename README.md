# Hyusap Tap

## How do I install these formulae?

`brew install hyusap/tap/<formula>`

Or `brew tap hyusap/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "hyusap/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
